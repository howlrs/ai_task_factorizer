# AI Task Factorization
現状を表すリソースを渡すとTodoを立ち上げ、各イシューに分類します。
現状を表すリソースは先方からのメールやドキュメント、問題提起文などを想定しています。

出力は非プログラマ向けに柔和な言葉での出力となっており、一般的に使いやすい文章での出力となっています。


## 環境変数
- GEMINI_MIDEL=<指定モデル>
- GEMINI_API_TOKEN=<xxx>
